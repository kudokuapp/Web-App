# WhatsApp Input

To import, use

```javascript
import WaInput from '$components/InputPlaceholder/WaInput';
```

**The input fills the whole container space.**

## Props

| Name        | Required | Type       | Description               |
| ----------- | -------- | ---------- | ------------------------- |
| id          | ✅       | String     | The ID of the input       |
| placeholder | ✅       | String     | Placeholder               |
| className   | ❌       | String     | Regular/Tailwind class    |
| value       | ✅       | String     | State value to store text |
| onChange    | ✅       | () => void | onChange handler          |

## Examples

```javascript
import WaInput from '$components/InputPlaceholder/WaInput';
import { useState } from 'react';

export const Example = () => {
  const [phoneNum, setPhoneNum] = useState('');

  return (
    <form
      onSubmit={(e) => {
        e.preventDefault();
      }}
    >
      <TextInput
        placeholder="WhatsApp kamu"
        value={phoneNum}
        onChange={(e) => {
          setPhoneNum(e.target.value);
        }}
      />
      <input type="submit" />
    </form>
  );
};
```
